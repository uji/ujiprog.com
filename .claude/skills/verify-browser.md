# ブラウザ確認

ローカル開発サーバーを起動し、変更内容をブラウザで確認する。

## 手順

1. `make run` をバックグラウンドで実行
2. 2秒待機後、`curl -s http://localhost:8788/` で変更を確認
3. 必要に応じて他のエンドポイントも確認（例: `/articles.json`）
4. `open http://localhost:8788` でブラウザを開く
5. 確認結果をユーザーに報告
6. 確認完了後、バックグラウンドのサーバーを停止
